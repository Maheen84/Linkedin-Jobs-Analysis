# Linkedin-Jobs-Analysis
Analyzed job data using Python, Pandas, and Seaborn to uncover trends in roles, skills, locations, and experience levels. Identified in-demand technologies and hiring patterns through EDA and visualizations, providing insights into current tech job market trends.


# 📊 LinkedIn Tech Jobs Analysis

## 🔍 Project Overview

This project analyzes LinkedIn tech job data to uncover trends in job roles, required skills, locations, and experience levels. The goal is to provide insights into the current tech job market using data analysis and visualization.

---

## 🎯 Objectives

* Identify most in-demand tech roles
* Analyze top required skills
* Explore job distribution by location and company
* Understand experience level requirements

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📁 Dataset Features

* Company Name
* Job Designation
* Location
* Experience Level
* Industry
* Skills (Python, SQL, Java, ML, etc.)
* Total Applicants

---

## ⚙️ Workflow

### 1️⃣ Data Cleaning

* Handled missing values
* Removed duplicates
* Ensured correct data types

### 2️⃣ Exploratory Data Analysis (EDA)

* Job roles distribution
* Top hiring companies
* Location-based analysis

### 3️⃣ Skill Analysis ⭐

* Identified most in-demand skills
* Analyzed skill requirements by role
* Compared technologies across job positions

### 4️⃣ Visualization 📈

* Bar charts for top skills & roles
* Pie charts for experience levels
* Count plots for job distribution

---

## 📊 Key Insights

* 🔹 Python, SQL, and Machine Learning are highly demanded
* 🔹 Certain cities dominate tech hiring
* 🔹 Entry and mid-level roles are most common
* 🔹 Skill requirements vary significantly by job role
